1) - Assure vous d'avoir NodeJS installé sur votre ordinateur et une version de PHP supérieure à  8.2.****


2) - Veuillez Modifier Les informations Dans le fichier .env pour votre sreveur MySQL

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=tsikonina
DB_USERNAME=root
DB_PASSWORD=



3) - Ouvrez dans unn terminal et lancer la commande 

npm run dev ( ne pas fermer, laisser le serveur tourner)



4) - Ouvrez dans un nouveau terminal et lancer la commande 

php artisan migrate (créér la base de donnée et les tables)



5) - Puis:

php artisan serve ( ne pas fermer, laisser le serveur tourner)
